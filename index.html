About

This is an extension to the javascript motion detection demo by Romuald.

It uses html 5 getUserMedia API and Web Audio API.

User's movement detection is done using a technique called blend mode difference.

Romuald has explained the concept and implementation of the demo in his blog and a more in-depth article on ADC.
